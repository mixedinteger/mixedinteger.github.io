Mixed Integer Programming Workshop 2023

May 22-25, 2023

University of Southern California

About

The Mixed Integer Programming (MIP) Workshop is a single-track workshop highlighting the latest trends in integer programming and discrete optimization, with speakers chosen by invitation.

The 2023 edition of the workshop will be the twentieth in the MIP series. Links to past editions can be found here.

Applications for student posters are now open! Please apply by completing this form.

Poster session

The 2023 MIP Workshop will include a poster session reserved for students. Please apply by completing this form.

The deadline for submission is March 1, 2023. Submissions should include an abstract in pdf format of at most 500 words.

Travel support will be offered to select students, both US-based and non-US-based.
