The following is the list of selected posters.

  • Alex Abreu (Federal University of São Carlos): Mathematical formulations for the multi-plant capacitated lot-sizing problem under stochastic demands
  • Agustina Anselmino (CONICET / UTN FRCU): Modeling Price Dynamics in Forest Biomass Biobasins via MIQP and Statistical Learning

  • Marcos Mansano Furlan (University of São Paulo): Product Placement Optimization in Supermarkets: Integrating Strategic and Tactical Shelf Decisions
  • Enzo Canesso Geraldini (Federal University of São Carlos): Robust Optimization applied to the Vehicle Routing Problem with Time Windows Multiple Deliverymen Customer Clustering

  • Valentina Moreno (Universidad Andrés Bello): Designing a Collaborative Municipal Recycling System via Capacitated Vehicle Routing for Cerro Navia and Quinta Normal (Chile)
  • Ignacio Nancupil (Universidad de Chile): An Enhanced Column Generation and Sharing Framework for Large-Scale Stochastic Power System Planning

  • Daniel Alfredo Neira (Virginia Tech): Multiple Shop Flow-Shop Problem with Lot Streaming
  • Matías Ortiz (Universidad de Chile): Multiple Matroid Secretary Problem
  • Matheus Jun Ota (University of Waterloo): Scenario-optimal recourse policies for vehicle routing problems with stochastic demands
  • Ieremies Vieira da Fonseca Romero (University of Campinas – UNICAMP): A Branch-and-Price Algorithm for the Graph Coloring Problem

  • Renan Fernando Franco da Silva (University of Campinas): A Combinatorial Branch-and-Bound Algorithm for the Bin Packing Problem

  • Alexander While (The Ohio State University): QUBO Approaches for MIP Symmetry Detection
